PiStream 

trait PiStream[T] extends PiPublisher[T] with HashSetPublisher[PiEvent[T]] with PiObservable[T]

A PiSource that also implements PiPublisher with Akka Streams. Uses akka.stream.scaladsl.BroadcastHub, which allows us to clone new sources and attach the handlers as sinks.
